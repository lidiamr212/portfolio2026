# Lidia Martínez - Porfolio/curriculum web

Una web estática diseñada y desarrollada desde cero como demostración de mi forma de entender el diseño digital.

El proyecto combina diseño editorial, UI y frontend para construir una experiencia donde la jerarquía visual, la legibilidad y la estructura del contenido son el eje principal.

---

## Tecnologías

| Capa | Tecnología |
|---|---|
| Estructura | HTML5 |
| Estilos | CSS3, CSS Variables, Flexbox |
| Interactividad | JavaScript (ES6) |
| Arquitectura CSS | CSS modular (Base · Components · Sections) |
| Tipografía | Google Fonts (Rubik) |
| Responsive | Mobile First + Media Queries |
| Animaciones | CSS Animations + JavaScript |
| Diseño | Figma |
| Control de versiones | Git + GitHub |
| Despliegue | GitHub Pages |

---

## Estructura del proyecto

```
porfolio/
├── css/
│   ├── reset.css             # Normalización de estilos
│   ├── variables.css         # Variables CSS (colores)
│   ├── typography.css        # Sistema tipográfico
│   ├── layout.css            # Layout y contenedores
│   ├── utilities.css         # Clases utilitarias
│   ├── components.css        # Componentes reutilizables
│   └── sections/
│       ├── hero.css
│       ├── info.css
│       ├── experienca.css
│       └── portfolio.css
│
├── js/
│   ├── hover.js                 # Acordeon efecto hover
│   ├── hero.js                  # Animación del Hero
│   ├── accordion.js             # Acordeón clasico
│   └── slider.js                # Galería y carrusel
│
├── img
│
├── index.html                   # Página principal
└── README.md                    # Documentación del proyecto
```
